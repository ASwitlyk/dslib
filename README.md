dslib
==============

A library of useful data structures in Javascript

## Installation
```
npm install dslib --save
```
## Usage
```
var dslib = require('dslib');

//create a stack for example:
var stack = new dslib.Stack();
```     
  

## Tests

```
gulp mocha
//or
gulp watch
```
